# Sad Cave — Project Overview

## Core idea
A moody social hangout game focused on atmosphere, emotion, and quiet interaction.

## Player fantasy
Players come here to relax, feel something, hang out, customize identity, and find small things to work toward.

## Current priorities
1. Retention / reasons to stay
2. UI clarity
3. Progression systems
4. Titles / identity / social flex
5. Monetization that fits the vibe

## Design rules
- Keep the vibe calm, moody, and premium
- Avoid loud or tacky systems
- Small safe changes over huge rewrites
- Preserve live systems unless explicitly targeted
