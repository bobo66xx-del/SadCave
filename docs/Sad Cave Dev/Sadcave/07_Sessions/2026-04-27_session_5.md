## 🐛 Broke / Fixed

- **XPBar invisibility (caught Cowork session 5 walkthrough).** Tyler reported the bar had vanished. Probe showed `XPBar` ScreenGui `Enabled=true`, all children `Visible=true`, `Fill` at 78.6% scale-X — bar IS rendering as a 4px semi-transparent strip at the bottom. Diagnosis: too subtle against dark cave backgrounds at 4px height. Fixed by PR #10 (height bump to 6px). The visual eventually still wasn't enough — required PR #11's contrast bump to fully resolve, but PR #10 was the first step.
- **AFK rate firing while seated (caught same walkthrough).** Captured live tick: `delta=+3, source=afk` while `Humanoid.Sit=true` and `SeatPart=Workspace.SeatMarkers.Seat`. Root cause: `PresenceTick.GetTickAmount` checked `isAFK` first, ignored `seatedAt`. Studio window losing focus to Cowork was triggering `AfkEvent(true)` via the focus-released hook. Fixed by PR #10 (Branch A — seated overrides AFK).
- **Console silence during XP ticks (caught same walkthrough).** Not a bug — XP system simply had no per-tick logging. Fixed by PR #10 (debug log).
