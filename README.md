# SadCave

SadCave is a Roblox social hangout experience built with Rojo.

## Project Structure

- `src/` — Rojo source tree
- `place-backups/` — backup place files
- `docs/` — project documentation
- `default.project.json` — Rojo project config

## Workflow

- Edit source in this repo
- Sync with Roblox Studio using Rojo
- Commit after each validated change
